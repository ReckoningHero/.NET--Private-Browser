bool DotNetBrowser.Events.StartLoadingArgs.IsSameDocument
get

Indicates whether the loading event is initiated by a navigation within the same document. Navigations within the same document do not change the DOM tree state. The following cases are considered the same document navigation:

  • Reference fragment navigations (e.g. http://www.example.org/foo.html#bar).
  • HTML5 History API: pushState() / replaceState() methods.
  • History navigation within the same page.

true if the navigation occurs within the same document.
