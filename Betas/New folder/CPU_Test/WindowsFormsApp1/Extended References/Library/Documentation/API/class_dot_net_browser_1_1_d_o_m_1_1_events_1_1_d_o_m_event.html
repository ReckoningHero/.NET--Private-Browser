This method must be called to set the event before it is dispatched, using EventTarget.dispatchEvent(). Once dispatched, it doesn't do anything anymore.

Parameters
eventTypethe type of DOM event
bubblesIs a Boolean deciding whether the event should bubble up through the event chain or not. Once set, the read-only property DOMEvent.Bubbles will give its value.
cancelableIs a Boolean defining whether the event can be canceled. Once set, the read-only property DOMEvent.Cancelable will give its value.
