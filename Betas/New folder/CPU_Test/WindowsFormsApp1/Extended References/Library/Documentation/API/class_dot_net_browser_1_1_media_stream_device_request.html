void DotNetBrowser.MediaStreamDeviceRequest.SetDefaultMediaStreamDevice ( MediaStreamDevice  device)

Updates default media stream device of the requested type. The device value must be the one from the list of available media stream devices you can obtain via the GetMediaStreamDevices() method.

Parameters
devicedefault media stream device. Cannot be null. It must be the one from the list of available media stream devices you can obtain via the GetMediaStreamDevices() method.
Exceptions
ArgumentExceptionwhen device is null, or its media type doesn't match the requested media type that can be obtained via the GetMediaStreamType() method, or the device isn't in the list of available devices.
